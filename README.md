# Bank Loan Data Analysis 🏦📊

## 📌 Project Overview
This project analyzes bank loan data to understand factors affecting loan approval and customer financial behavior using data analysis techniques.

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## 📊 Project Workflow
- Data Cleaning: Handled missing values, duplicates, and inconsistent data.
- Exploratory Data Analysis (EDA): Explored relationships between income, credit history, loan amount, and approval status.
- Data Visualization: Built charts and graphs to present financial insights.

## 🔍 Key Insights
- Factors influencing loan approval decisions.
- Impact of credit history and income on loan status.
- Distribution patterns of loan applicants.

## 🎯 Objective
To analyze loan application data and extract insights that support better financial decision-making.
